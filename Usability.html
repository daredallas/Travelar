Usability Goals

Able to provide specific user interfaces for different situations (keyboard, star rating, directions)

Controls can be used in both landscape and portrait orientation

Able to preload nearby location data when the app is opened

Able to provide relevant data for locations within a second (if not preloaded)

Able to collect user information anonymously to improve services

New users able to find their first interactive location within 30 seconds of opening the app
